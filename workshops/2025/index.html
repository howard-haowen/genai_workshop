四週課程大綱：AI 輔助數位創作與公開發表

本微學程單元以「文科視角的 AI 應用開發入門」為核心，設計為四週循序漸進的實作課程。課程目標在於培養學生運用 Markdown 結構化寫作、Git/GitHub 版本管理、GitHub Copilot 與 Codespaces AI 輔助開發、GitHub Pages 與 Actions 自動化部署 的能力，使非資訊背景學生也能完成可公開展示的數位作品。課程採講解、示範、任務導向實作與成果發布並行的方式，協助學生建立從內容發想、資料組織、AI 協作、互動網頁製作到線上發布的完整創作流程。

  • 第 1 週：以 Markdown 建立專案筆記與內容結構，搭配 Git/GitHub 理解版本管理、提交紀錄與協作保存。
  • 第 2 週：導入 GitHub Copilot 與 Codespaces，練習以提示語生成網頁元件、程式片段與創作素材。
  • 第 3 週：比較 Copilot Ask、Edit、Agent 三種互動模式，完成如台語單詞卡等小型 AI 輔助應用改作。
  • 第 4 週：使用 GitHub Pages 與 GitHub Actions 發布作品，理解靜態網站部署、自動化轉檔與成果展示流程。

2025 材料現況

4 份週課程簡報 W1–W4：從 Markdown/Git 到 GitHub Pages 與 Actions。 2 份補充簡報 Markdown 基礎與 Git 基礎，搭配課堂實作使用。 3 個應用程式範例 台語單詞卡、井字遊戲、個人網站範例。 Source 講義與筆記 Markdown 講義、部署問答、Kanban 與轉檔腳本保留在 GitHub 原始碼中。
