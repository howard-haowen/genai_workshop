AI創造工作坊

2026 年度課程面向台灣大專院校人文學科學生，核心不只是學會使用 AI 工具，而是成為能設計、調整、分享 AI 工作流程的 AI 創造者。課程將引導學生把閱讀、寫作、研究、翻譯、整理與創作任務，轉化為可重複、可改良、可分享的 AI-assisted workflow。

五週課程大綱：從 AI 使用者到 AI 工作流程創造者

本課程以人文學科的學習、研究與創作需求為出發點，帶領學生理解 AI agent、tools、AGENTS.md、skills、SKILL.md、MCP、Git/GitHub、GitHub Pages、Obsidian 與 Markdown 如何共同構成可操作的個人 AI 工作流程。完成課程後，學生將能把一個日常學習或創作任務，整理為具備專案脈絡、操作規則、技能規格與公開展示頁的 personal AI-assisted workflow。

  • 第 1 週：從 AI 使用者到 AI 創造者，盤點人文學科常見任務，建立 Obsidian 筆記資料夾，並產出 workflow_idea.md。
  • 第 2 週：理解 AI Agent 與 Tool 的差異，將個人任務拆解為 agent steps，建立 GitHub repository，並產出 workflow_steps.md。
  • 第 3 週：用 AGENTS.md 建立專案脈絡、語言偏好、工作規則與驗證清單，測試 AI agent 是否能依規則穩定執行任務。
  • 第 4 週：認識 Skills for AI Agents 與 SKILL.md 結構，將可重複任務封裝為 skill 草稿，並在 IBM Bob 中測試指令是否清楚。
  • 第 5 週：理解 MCP 如何連接外部工具與資料，整理五週產出，使用 GitHub Pages 發布個人 AI-assisted workflow 展示頁。
